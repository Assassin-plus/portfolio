# 活字生形

我的主要贡献是人工智能光学字符识别（OCR）插件系统。我将最先进的开源模型PPOCR-v5集成到虚幻引擎中，支持核心游戏玩法。

除此之外，作为一名技术艺术家，我还帮助创作漫画风格化的着色器、视觉效果、环境艺术设计和所有照明。
