# Abstract

Automated generation of large-scale 3D scenes presents a significant challenge due to the required training resources and datasets for 3D scenes. This is in sharp contrast to the 2D counterparts that has become readily available due to its superior speed and quality. However, a line of prior works in 3D on procedural modeling has shown promise in generating high-quality assets using the combination of algorithms and rules based on user inputs. To leverage the best of both 2D generative models and large-scale procedural modeling tools, we present TerraCraft, a novel algorithm for generating high-resolution 3D geospatial scenes. Using the power of Large Language Models (LLMs), TerraCraft can generate large-scale 3D scenes from natural text descriptions from the users. With a procedural modeling approach, TerraCraft bridges the gap between sophisticated diffusion model-based 2D image generation and commercial 3D scene generation. With its intuitive interface and powerful capabilities, TerraCraft enables users to easily create high-resolution detailed scenes to be used in various applications, such as virtual reality, game design, and visual datasets. We demonstrate the effectiveness of TerraCraft through extensive experiments and user studies, showing that TerraCraft can efficiently generate detailed, high-quality 3D scenes.
